## 描述

这是一个专用脚本，仅根据缓冲区面积执行缓冲区分析。它围绕源点构建多环缓冲区，并计算每个缓冲环与指定区域边界相交的面积。

## Toolbox script Structure

该工具需要以下 **[6 个主要输入字段](#输入参数)**：
![Photo](../images/buffer_Analysis_Area.png)

## 输入参数

### 源兴趣点（POI）图层

此字段指定包含源兴趣点（POI），既缓冲区中心的点图层。

### 选择源兴趣点（POI）的区域名称字段

此字段指定为源兴趣点定义区域名称（例如行政区）的字段。

### 区域图层

此字段指定定义每个区域边界的多边形图层。

### 选择区域图层的区域名称字段

此字段指定为多边形图层定义区域名称的字段。

> [!NOTE]
> 确保 **[选择源兴趣点（POI）的区域名称字段](#选择源兴趣点（POI）的区域名称字段)** 和 **[选择区域图层的区域名称字段](#选择区域图层的区域名称字段)** 中的值对应相同的区域。强烈建议使用 [District Divid 脚本](../district_divid) 从 **[区域图层](#区域图层)** 为 **[源兴趣点（POI）图层](#源兴趣点（POI）图层)**生成区域名称字段。

### 定义缓冲区（米）（Python）

此字段使用 Python 列表或列表推导式定义缓冲区半径（以米为单位）。
> [!EXAMPLE]
>
> 1. **定义特定的缓冲区距离（例如 10、20、50、100、500、1000 米）**
>
>     ```python
>     [10, 20, 50, 100, 500, 1000]
>     ```
>
> 2. **生成从 10 米到 2000 米、间隔为 10 米的缓冲区环**
>
>     ```python
>     [10 * x for x in range(1,201)]
>     ```

## 结果

结果包含四列：

- **city**: 区域名称，由 **[源兴趣点（POI）图层](#源兴趣点（POI）图层)**定义.
- **distance**: 缓冲区半径（以米为单位）。
- **Num**: 对于相应的 **city** 和 **distance**，缓冲环与相应城市边界相交的面积（以平方米为单位）。
- **totalNum**: 相应 **city** 的总面积（以平方米为单位）。
